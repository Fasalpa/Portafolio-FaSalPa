# 🌿 Fasalpa | Personal Portfolio

Bienvenido a mi portafolio personal. Este proyecto reúne mi experiencia, formación y proyectos como desarrollador Full Stack Junior (sin olvidar todo el feedback que he pedido a mis preciadas personas cercanas).

Fue diseñado y desarrollado desde cero con un enfoque en la experiencia de usuario, la accesibilidad y un diseño limpio que refleja mi forma de trabajar.

## 🌐 Demo

🔗 **Sitio web:** https://fasalpa.github.io/Portafolio-FaSalPa/

---

# ✨ Características

- 🌙 Modo claro y modo oscuro.
- 🌍 Internacionalización (Español / English).
- 📱 Diseño completamente responsive.
- ♿ Enfoque en accesibilidad mediante etiquetas ARIA y navegación intuitiva.
- 📄 Descarga dinámica del CV según el idioma seleccionado.
- 🎨 Diseño moderno con identidad visual propia.
- ⚡ Interfaz optimizada para una experiencia fluida.
